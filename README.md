## Run Locally

```bash
npm install
npm start
```

Then open `http://localhost:3000` in your browser.

## Deploy to Render

1. Push this repo to GitHub
2. Go to https://render.com
3. Create a new "Web Service"
4. Set:
   - **Start Command**: `npm start`
   - **Build Command**: `npm install`
   - **Root Directory**: `.`
   - **Node Version**: 18+
